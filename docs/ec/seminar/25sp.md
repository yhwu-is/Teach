# 2024-2025 学年春夏学期计算经济研讨

## 整体计划

- **时间**：1-15 周，每周五下午 13:30-17:30

- **地点**：玉泉校区教 4-124

- **讨论主题安排**：
    - 博弈论与计算经济专题 主讲人：吴一航
    - 均衡计算专题 主讲人：金政羽
    - 拍卖理论专题 主讲人：阳先毅
    - 算法博弈论专题 主讲人：郑涵文

- **参考材料**：

    - 《博弈论》【以】迈克尔·马希勒，埃隆·索兰，什穆埃尔·扎米尔

    - 《博弈论与信息经济学》张维迎

    - 《算法博弈论二十讲》【美】Tim Roughgarden
    - 《Algorithmic Game Theory》【以】Nissan 等
    - 其它相关教材，课程讲义 / slides，论文等

- **预修要求**：
    - 微观经济学与博弈论基础
    - 可能被遗忘的基础线性代数、微积分、概率论以及算法分析知识
